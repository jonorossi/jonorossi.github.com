Castle Visual Studio Integration

Castle Visual Studio Integration (CVSI) provides Microsoft Visual Studio 2005/2008/2010 integration for the Castle Project. Currently it only contains a colorizer and IntelliSense for the NVelocity language.

New Project Wizard: If you are looking for the New Project wizard you will need the official Visual Studio integration. See the Castle Project web site for more details.

Having trouble with syntax hightlighting not working?

Visual Studio (especially 2012+) seems to aggressively cache its fonts and colours cache. You’ll need to manually reset the cache by deleting the Cache key in the registry:

HKEY_CURRENT_USER\Software\Microsoft\VisualStudio\xx.0\FontAndColors\Cache
